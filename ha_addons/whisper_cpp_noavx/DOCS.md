# Sokol-9 Whisper.cpp NoAVX

Локальный Wyoming-сервер распознавания речи для Сокола-9. `whisper.cpp`
компилируется на сервере Home Assistant с отключённым автоопределением SIMD,
поэтому бинарник не требует AVX, AVX2, FMA или F16C.

Сервис слушает TCP-порт `10301` на хосте Home Assistant. По умолчанию
используется русскоязычная модель `base-q5_1` с `beam_size: 1`.
