# Сегодняшний домашний тест Сокола-9

Цель: за два часа установить аддон и пройти один сквозной сценарий.

## Нужно принести

- доступ к Home Assistant Supervisor;
- OpenAI API key и небольшой предоплаченный баланс с месячным пределом;
- entity ID камеры, док-станции, моторов и media player;
- по три чистых голосовых образца Виктора, жены и дочки;
- несколько кадров каждого человека при обычном домашнем освещении;
- свободный безопасный участок пола до зарядной базы.

## Приёмка

1. Сокол различает Виктора, жену и дочку либо честно сообщает низкую уверенность.
2. Виктор ведёт с ним короткий связный разговор; голосом остаётся только Sol.
3. По команде «следуй за мной» робот медленно сопровождает Виктора и останавливается при потере цели, препятствии или ошибке датчика.
4. По команде «едь домой» сопровождение прекращается, робот возвращается на базу.
5. Только после подтверждённой стыковки камера выключается, обработка речи останавливается и состояние становится `sleeping`.
6. В HA видны состояние, распознанный говорящий, события безопасности и расход API.

Не считать тест пройденным, если личность угадана без достаточной уверенности, движение продолжилось после потери человека либо камера выключилась до подтверждения зарядки.
