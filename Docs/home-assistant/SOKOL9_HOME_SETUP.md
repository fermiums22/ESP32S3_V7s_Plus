# Быстрый запуск Сокол-9 дома

## Что потребуется

- домашний Windows-компьютер с этим репозиторием;
- доступ к Home Assistant по локальной сети;
- доступ к сетевой папке `\\homeassistant\addons` или возможность загрузить ZIP;
- отдельный OpenAI API key для домашнего агента;
- long-lived access token Home Assistant только на время диагностического probe.

Токены и пароли не записывать в репозиторий и не присылать в чат.

## 1. Собрать сведения о доме

В профиле Home Assistant создать long-lived access token. В новом `cmd.exe`:

```bat
set SOKOL9_HA_TOKEN=вставить_токен_локально
probe_home.bat
set SOKOL9_HA_TOKEN=
```

Результат появится в `tools/sokol9_home_probe.json`. Он содержит только краткие
сведения о нужных сущностях и сервисах. По нему выбираются Whisper, Supertonic,
JBL, GoPro, Xiaomi X20+, карта, датчики присутствия, телевизоры и пульты.

## 2. Установить add-on

Если доступна Samba-папка HA:

```bat
install_ha_addon.bat
```

Скрипт также копирует пакет автоматизаций в `\\homeassistant\config\packages\sokol9.yaml`.
В `configuration.yaml` должно быть включено:

```yaml
homeassistant:
  packages: !include_dir_named packages
```

Если папка называется иначе, передать её параметром `-HaAddonsPath`. Альтернатива:

```bat
package_ha_addon.bat
```

ZIP будет создан в `dist/sokol9-ha-addon.zip`.

В Home Assistant открыть **Настройки → Приложения → Магазин приложений**, обновить
локальные приложения, установить или пересобрать **GoPro Assist Bridge**.

## 3. Первые настройки

До первого запуска проверить:

- `rtsp_url` — поток GoPro `gopro_robot`;
- `stt_engine` — локальный `stt.whisper_cpp` из add-on `Sokol-9 Whisper.cpp NoAVX`;
- `tts_entity` — фактическая сущность Supertonic;
- `media_player` — JBL робота;
- `wake_phrases` — `сокол девять,сокол`;
- `camera_entity` — `camera.robot_eyes`;
- `home_map_entity` — найденная карта Xiaomi X20+;
- `openai_api_key` — отдельный ключ домашнего проекта;
- `agent_enabled` — включить после проверки остальных полей.

Начальные лимиты: `$0.25` в день, `$3` в месяц, до 250 выходных токенов.

## 4. Проверка

После запуска должны появиться:

- `sensor.gopro_assist_status`;
- `sensor.gopro_assist_transcript`;
- `sensor.gopro_assist_response`;
- `sensor.robot_agent_input_tokens`;
- `sensor.robot_agent_output_tokens`;
- `sensor.robot_agent_requests_today`;
- `sensor.robot_agent_cost_today`;
- `sensor.robot_agent_cost_month`.

Проверочные фразы:

1. `Сокол, повтори: проверка связи.`
2. `Сокол, какое состояние робота?`
3. `Сокол, посмотри перед собой.`
4. `Сокол, найди карту дома.`

## 5. Калибровка дистанции и сегодняшний тест

Сначала прошить STM32 (протокол Modbus 7), затем ESP32-S3. Поставить перед
роботом плоскую матовую поверхность ровно в 30 см и один раз нажать в HA кнопку
`Calibrate front IR at 30 cm`. Сырые ИК-отсчёты наружу не публикуются; появятся
`Front distance left/center/right` в сантиметрах. Из-за оптического принципа цвет
одежды влияет на оценку, это не ToF-лидар.

Каждый человек трижды, стоя примерно в метре от робота, говорит одну и ту же
фразу со своим именем:

1. `Сокол, запомни мой голос я Виктор.`
2. `Сокол, запомни мой голос я Мама.`
3. `Сокол, запомни мой голос я Дочка.`

Проверять по `sensor.sokol_9_speaker`. Это бытовое различение, не средство
аутентификации.

Проверка движения проводится на свободном полу, сначала рукой над аварийной
кнопкой:

1. `Сокол, следуй за мной.` — человек остаётся перед роботом; скорость ограничена
   22 rpm, при потере цели робот тормозит и через 3 секунды выходит из режима.
2. `Сокол, стой.` — обе цели колёс обнуляются и привод отключается.
3. `Сокол, езжай домой.` — локальная команда без OpenAI вызывает стыковку.
4. Через 5 секунд после `Robot docked = on` поток GoPro останавливается, камера
   получает команду sleep.

В HA создаётся `vacuum.sokol_9`: его `return_to_base` вызывает тот же сценарий.
Эту сущность можно экспортировать в интеграцию Алисы как отдельного робота.

## После первого теста

Карта X20+ уже имеет комнаты `Спальня`, `Офис`, `Коридор`, `Детская`, `Гостиная`,
`Кухня`, `Главный вход`; второй робот называется **Мика**. Probe найдёт её
`vacuum`/`image` сущности. Координатная привязка карты, визуальное сопровождение
конкретного лица, автономное прохождение порогов и каталог японских штатных фраз
Мики подключаются после безопасной проверки сегодняшнего контура.

Для подготовки визуального каталога в каждой комнате сделать 4–8 остановок с
разным направлением камеры и сказать `Сокол, запомни место <название>`. Желательно
повторить несколько кадров вечером. Команда `Сокол, где мы?` проверяет локальное
сравнение; кадры и одометрия сохраняются в данных add-on и не расходуют API.
